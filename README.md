# Компіляція скінченного автомату для regex-виразу
Цей репозиторій містить реалізацію лабораторної роботи із дискретної математики.

## Коротке пояснення імплементації
Реалізовано класи, які відповідають за різні типи станів (початковий, заключний, *, +, . , Ascii letters or digits).

Головною частиною є клас RegexFSM, що компілює regex-вираз у скінченний автомат, а також має метод check_string, що перевіряє чи може автомат прочитати наданий ланцюжок.


## Інструкція до запуску

Ви можете використовувати цей проєкт як бібліотеку у своєму коді або як консольну програму через argparse.

### 1. Використання як модуля

Імпортуйте клас у свій Python-скрипт для програмної перевірки рядків:

```python
from regex import RegexFSM

# Ініціалізація автомата
regex_pattern = "a*b+"
fsm = RegexFSM(regex_pattern)

# Перевірка рядка
test_string = "aaabb"
print(fsm.check_string(test_string))
```

### 2. Використання через консоль

Приклад запуску у консолі:

python regex.py <regex-expr> <string_to_check>

```python
python regex.py "a*b" "aaab"
```
