**Компіляція скінченного автомату для regex-виразу**
Цей репозиторій містить реалізацію лабораторної роботи із дискретної математики.
**Коротке пояснення імплементації**
Реалізовано класи, які відповідають за різні типи станів(початковий, заключний, *, +,., Ascii letters or digits.
Головною частиною є клас RegexFSM, що компілює regex-вираз у скінченний автомат, а також має метод check_string, що перевіряє чи може автомат прочитати наданий ланцюжок.
**Інструкція до запуску**
Без argparse
1.Імпортувати клас RegexFSM з файлу regex.py
2. Створити екземпляр класу RegexFSM
3. Використовувати метод check_string для перевірки чи читає створений автомат наданий ланцюжок
З argparse
python regex.py <regex-expr> <string_to_check> <string_to_check>
